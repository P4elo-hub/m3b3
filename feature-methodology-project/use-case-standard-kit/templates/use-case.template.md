**Основной сценарий — Happy Path:**

```
Шаг 1:  [Актор] [атомарное действие]
Шаг 2:  Система [атомарная реакция]
          ЕСЛИ [BR-01] [проверяемое условие]
            ТО сценарий продолжается с шага 3
          ИНАЧЕ
            ПЕРЕЙТИ К альтернативному сценарию 2a
Шаг 3:  Система [следующее действие]
```

**Альтернативные сценарии:**

```
2a. [Условие ветвления] ([Тип альтернативы]):
  2a.1. Система [действие]

```
